# Discord動画圧縮アプリ — ビルド手順

## これは何？
動画ファイルをDiscordで送れる10MB以下に圧縮するアプリです。
**FFmpegはアプリに内蔵**されているので、ユーザーは別途インストール不要です。

---

## .exe を作る手順（開発者向け）

### 必要なもの
- Node.js 18以上: https://nodejs.org/ja

### 手順

```bash
# 1. このフォルダに移動
cd video-compressor-v2

# 2. パッケージをインストール（FFmpegも自動でダウンロードされます）
npm install

# 3. ビルド → dist/ フォルダに exe が生成されます
npm run build
```

`dist/Discord動画圧縮.exe` が完成します。  
このファイル1つを配布するだけでOK。受け取った人はそのまま使えます。

---

## アプリの使い方（ユーザー向け）
1. `Discord動画圧縮.exe` をダブルクリックして起動
2. 動画ファイルをウィンドウにドロップ（またはクリックして選択）
3. 目標サイズを選択して「圧縮スタート」
4. 完了したら動画ファイルと同じフォルダに `_compressed.mp4` が保存されます
